# IndoWeatherSensor

Website penjualan Weather Station berbasis PHP Native dan MySQL.

## Fitur

- Login & Register
- Role Admin dan User
- CRUD Produk
- Upload Gambar dan Datasheet
- Weather Station Configurator
- Simpan Inquiry
- Dashboard Admin
- Kelola Inquiry
- Edit Profil
- Ganti Password
- Pencarian Produk
- Responsive Bootstrap 5

## Teknologi

- PHP Native
- MySQL
- Bootstrap 5
- HTML5
- CSS3
- JavaScript
- Font Awesome

## Database

Terdiri dari 5 tabel utama:

- users
- kategori
- produk
- konfigurasi
- konfigurasi_detail

## Cara Instalasi

1. Clone repository

```bash
git clone https://github.com/MNAdamB/IndoWeatherSensor.git
```

2. Import database (extact dulu file zipnya)

```
database.zip
```

3. Jika runnging di hosting sendiri, Edit file ini untuk koneksi db

```
koneksi.php
```

4. Jika running di localhost, Edit file ini untuk path directory

```
config.php
```
5. Jalankan menggunakan XAMPP

```
http://localhost/web_lanjutan/indo_weather
```
